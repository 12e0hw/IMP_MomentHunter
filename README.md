# Moment Hunter
2025-1 Immersive Media Programming VR Team Project

Unity로 제작한 VR 서사형 사진 퍼즐 게임입니다.

플레이어는 VR 환경에서 카메라를 사용해 미션 단서에 맞는 장면을 촬영하고, 사진을 통해 가족의 추억과 숨겨진 이야기를 발견하며 게임을 진행합니다. 

제한된 필름 수 안에서 주변 오브젝트를 관찰하고 배치하며, 미션 조건에 맞는 사진을 찍는 것이 주요 목표입니다. 플레이어는 3D 공간 안에서 이동과 상호작용을 통해 단서를 찾고, 촬영한 사진에 포함된 오브젝트를 기반으로 미션 성공 여부를 판정받습니다. 


### Tech Stack
- Engine: Unity
- Language: C#
- Framework: XR Interaction Toolkit
- Platform: Meta Quest 2(VR)
